# Guide d'Installation - Extract.io pour macOS 🍏

Félicitations ! Vous disposez de la version autonome d'**Extract.io**. Ce guide vous explique comment l'installer et l'utiliser sur votre Mac.

---

## 🚀 Installation Rapide

1. **Extraction** : Double-cliquez sur le fichier `Extract.io-1.0.0-mac.zip` pour le décompresser.
2. **Installation** : Faites glisser l'icône **Extract.io.app** dans votre dossier **Applications**.
3. **Ouverture** :
   - Faites un **clic-droit** sur l'application et choisissez **Ouvrir**.
   - Si macOS affiche un message de sécurité (développeur non identifié), cliquez sur **Ouvrir** à nouveau.
   - *Note : Cela n'est nécessaire que lors du premier lancement.*

---

## 🎯 Comment ça marche ?

Une fois l'application lancée, tout est automatique :
- **Moteur intégré** : Le moteur d'extraction démarre tout seul en arrière-plan.
- **Interface intuitive** : Utilisez l'onglet **Unique** pour un site ou **Bulk** pour une liste.
- **Résultats** : Les données s'affichent sous forme de fiches. Cliquez sur le bouton **CSV** pour tout exporter vers Excel ou Google Sheets.

---

## 🛠 En cas de problème

- **L'application ne s'ouvre pas** : Allez dans `Réglages Système` > `Confidentialité et sécurité` et cliquez sur "Ouvrir quand même" en bas de la page.
- **Aucune info trouvée** : Vérifiez votre connexion internet. Certains sites très protégés peuvent mettre quelques secondes de plus à être analysés par notre intelligence hybride.
